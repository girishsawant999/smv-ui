## CSS Variables

```css
body {
  // Fonts Family
  --primary-font-light: 'Manrope-ExtraLight';
  --primary-font-extra-light: 'Manrope-ExtraLight';
  --primary-font-medium: 'Manrope-Regular';
  --primary-font-regular: 'Manrope-Regular';
  --primary-font-bold: 'Manrope-Regular';
  --primary-font-semi-bold: 'Manrope-SemiBold';
  --primary-font-extra-bold: 'Manrope-ExtraBold';
  --secondary-font: 'Sequel-76';

  // Colors
  --color-black: #000;
  --color-white: #fff;
  --color-black-primary: #1a181b;
  --color-black-secondary: #4e4851;
  --color-black-tertiary: #9b939f;
  --color-grey-primary: #f5f4f5;
  --color-red: #ea4848;
  --ripple-effect-color: #7f7f7f7f;

  --box-shadow-default: 0px 0px 20px rgba(0, 0, 0, 0.03), 20px 20px 40px rgba(0, 0, 0, 0.03);
  --box-shadow-button: 0 0 0 2px #e6e4e7;

  --border-width: 2px;
  --border-radius: 1.25em;

  --transition-duration: 200ms;
  --transition-function: ease-in-out;
  --transition: var(--transition-duration) var(--transition-function);
}
```
